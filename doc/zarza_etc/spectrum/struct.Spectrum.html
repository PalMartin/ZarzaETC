Struct Spectrum

Source 
pub struct Spectrum { /* private fields */ }
Expand description

A generic spectrum structure built upon a Curve structure. The “spectrum” encapsulates a “Curve” and provides additional operations for spectra, such as scaling by a factor or another curve, and axis inversion consistent with physical units. Spectra are commonly used to represent distributions (e.g., intensity, energy) across a domain such as wavelength or frequency. This struct enables both general curve manipulation and domain-specific spectral operations.
